### Dependencias

- Chakra
- Axios
- React icons
- React router dom
- React spinners
- React toastify
- SweelAlert2
- Giphy


### Instalacion

- Clonar repositorio
    - git clone 
- Ingresar a la carpeta
    - cd [carpeta]
- Instalar las dependencias
    - npm install
- Compilamos proyecto
    - npm run dev
