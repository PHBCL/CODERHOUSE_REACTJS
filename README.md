### Instalacion

- Clonar repositorio
    - git clone https://github.com/PHBCL/CODERHOUSE_REACTJS.git
- Ingresar a la carpeta
    - cd CODERHOUSE_REACTJS
- Instalar las dependencias
    - npm install
- Compilamos proyecto
    - npm run dev
